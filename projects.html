P R O J E C T S

IBM Data Science Capstone project: Space X Falcon 9 landing predictions

This capstone project was aimed to create classification models that could accurately predict the successful landing of the Falcon 9 first stage for SpaceX rocket launches. By exploring various factors that might affect the landing outcome, predictive models were developed that provided valuable insights for competing companies interested in bidding against SpaceX for rocket launches. This project revolved around uncovering the essential determinants of successful rocket landings.

Prediction Analysis on Student's Dropout

The aim of this project is to create a predictive model using machine learning classification algorithms to detect students at risk of dropping out. By utilizing student data such as demographics, academic performance, socio-economic factors, and other relevant features.

Web Scrapping and Python Data visualization on Top 500 movies IMDb

The primary objective of this project was to extract valuable information from movie rating website, e.g. IMDb, using webscraping techniques and uncover hidden patterns and reveal captivating insights based on movie features like ratings, genres, director information, release dates, production years, and more.
