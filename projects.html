IBM Data Science Capstone project: Space X Falcon 9 landing predictions

This capstone project aimed to develop classification models capable of accurately predicting the success or failure of SpaceX's Falcon 9 first stage landings. By analyzing various factors influencing the landing outcome, predictive models were created that provided valuable insights for competing companies aiming to bid against SpaceX for rocket launches. This project focused on identifying the crucial determinants of successful rocket landings.

Prediction Analysis on Student's Dropout

The aim of this project is to create a predictive model using machine learning classification algorithms to detect students at risk of dropping out. By utilizing student data such as demographics, academic performance, socio-economic factors, and other relevant features.

Web Scraping and Python Data visualization on Top 500 movies IMDb

The primary objective of this project was to extract valuable information from movie rating websites, e.g. IMDb, using web scraping techniques and creating an interactive dashboard captivating insights based on movie features like ratings, gross earning, and more.
